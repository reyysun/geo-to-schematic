GeoToSchematic
Convert KML and GeoJSON files into Minecraft schematics using BuildTheEarth projection!
Elevation handling is supported.
Import your file:

    ❔ The block used for lines and points in the schematic.
    ❔ On some BTE servers, the Y level may not match the actual elevation.
    You can set how much objects will be offset vertically from the Y levels they're supposed to be on.
    For example, this value has to be "-2016" for Italy schematics, because Y level -2016 equals real elevation 0 on BTE Italia.
    ❔ If disabled, the vertices of lines and polygons will not be connected and they'll become points.
    Privacy notice: all files are processed by your local machine and aren't uploaded to any server.
